	AND WHAT IMPORTANT INFO WE NEED ABOUT EACH.

	IsMajor		SPECIFIES WHETHER IT IS A MAJOR KEYWORD
	HasMinors	SPECIFIES IF THE KEYWORD HAS MINOR KEYWORDS
	DataDict	SPECIFIES THE DATA DICTIONARY TO USE
	ArgTypes	IS A STRING LIST THAT DESCRIBES THE
				ORDER AND TYPE OF ARGUMENTS

		B	=	BOOLEAN	(USED FOR FILETYPE:)
		I	=	IGNORE	(USED FOR SET: AND EVAL:)
